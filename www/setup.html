S2Container.NETはMicrosoft .NET Framework 1.1上で動作します。 稼動確認は、Microsoft Visual Studio .NET 2003で行っています。

s2container.net-version.zip を解凍してできたディレクトリにあるsource/Seasar.slnファイルを、 Microsoft Visual Studio .NET 2003で開くとS2Container.NETのソースを見ることができます。 Examplesやテストコードも含まれています。 テストにはNUnitを使用しています。

S2Container.NETを利用する場合には以下のファイルの参照を追加する必要があります。 なお、libディレクトリにあるSeasar.dllとja-JPディレクトリは同じディレクトリに配置してください。

  • lib/Seasar.dll
  • lib/log4net.dll

Examplesを実行するには、Visual Studio .NET 2003でsource/Seasar.slnを開き、 Seasar.Examplesプロジェクトをスタートアッププロジェクトに設定して下さい。 実行すると、ExamplesExplorerが起動しS2Container.NETの機能を体験することができます。

Examplesでは一部、SQL Server 2000（もしくは MSDE 2000）を必要とする機能があります。 その機能を試す場合には、data/setUpDemo.batを実行し、Examples用のデータベースを作成します。 そして、Seasar.ExamplesのTx.diconとDTCTx.diconのデータベースへの接続文字列を お使いの環境に合わせて変更して下さい。
