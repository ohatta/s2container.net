Introduction

S2Container.NET uses JScript.NET. Script used in a configuration file may be converted into a .NET object.

  • String are surrounded by "'s. Example: "hoge"
  • char are surrounded by '. Example: 'a'
  • Integer are written as they are. Example: 123
  • System.Boolean have values true and false
  • System.Enum are written with a fully qualified name and member name. (This is noot in strictly accordance with JScript.NET convention.) Example: System.Data.DbType.Decimal
  • System.Type is written as fully qualified name. (This is noot in strictly accordance with JScript.NET convention.) Example: Foo.Bar
  • Standard library (mscorlib.dll), class, and constructors may be called. (System in the namespace is omitted.) Example: new DateTime()

Refer to JScript.NET Reference for further details.

Following JScript.NET variables may be used in the configuration file:

Name Description
self Variable defined only in body of <initMethod> or <destroyMethod>. This variable references the current component (object within <component> elements in <initMethod> or <destroyMethod>).

Example

This example uses JScript.NET in the body of the property tag.

String

<property name="Message">"Hello"</property>

Value

<property name="Pi">3.14</property>

S2Container

This example retrieves value assigned to the key "Hello" from the Hashtable named "Message" in the S2Container. 

<property name="Message">container.GetComponent("Message")["Hello"]</property>

Application Configuration File

This example retrieves value assigned to the key "message" when application configuratin file is defined as below:

<configuration>

    <appSettings>
        <add key="message" value="Hello" />
    </appSettings>
    
</configuration>

<property name="Message">appSettings['message']</property>
