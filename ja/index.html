概要

上へ

　S2Container.NETは、AOP(Aspect Oriented Programming)をサポートしたDI(Dependency Injection)コンテナです。 .NET Framework環境で動作します。最新バージョンは.NET Framework 2.0と.NET Framework 1.1に対応しています。

　DIとAOPを扱い品質の高いソフトウェアを効率よく開発する為に開発されたS2Container.NETは、 テストを楽しく簡単に行えるようにS2Unit.NETというテスティングフレームワークや、 Windowsアプリケーション開発をサポートするS2Windows.NETの提供も行っています。

　近日中には、業務アプリケーションの開発に手軽にDIとAOPを導入することができるQuillという簡単DI機能の提供を予定しています。

　また、S2Dao.NETと共に利用することで、データアクセス層の生産性をさらに向上させることができます。
